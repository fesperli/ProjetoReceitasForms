# Gerenciador de Receitas

Projeto desenvolvido para a disciplina de **Desenvolvimento Desktop**, solicitado pelo professor **Lucas Dalilo**.

## Descrição do Projeto

O **Gerenciador de Receitas** é uma aplicação desktop desenvolvida em **C# com Windows Forms**, com o objetivo de permitir o cadastro, consulta, edição e exclusão de receitas culinárias.

O sistema permite gerenciar receitas, categorias, ingredientes, usuários e avaliações. Os dados são armazenados em um banco de dados relacional SQL Server, garantindo que as informações cadastradas permaneçam salvas mesmo após o fechamento da aplicação.
